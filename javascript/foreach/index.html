Looping through javascript arrays with Array.forEach()

This tutorial will demonstrate how to use the Array.forEach() method with some examples

The forEach() method can be called with any javascript array, and will let you run a function for each element in the array.

First, let's take a look at the parameters passed in to the Array.forEach() method.

Array.forEach( function(current_value, index, initial_array) {}, this_context )

Since the callback function gives us access to the current value, index, and initial array; the forEach() method is typically used to traverse an array performing some task on each element.

this_context is an optional parameter, and a more in depth explanation can be found at the following Array.prototype.forEach() MDN article https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach

So, let's take a look at a simple example of using forEach() to total all of the numbers in an array

			
				var sum_of_numbers = 0;

				var array_of_numbers = [5, 7, 1, 9, 8, 5];

				array_of_numbers.forEach(function(current_value, index, initial_array) {
					sum_of_numbers += current_value;
				});

				console.log(sum_of_numbers);
				//35
