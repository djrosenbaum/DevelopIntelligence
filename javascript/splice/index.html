Unit conversion with javascript Array.map()

When you already have an existing array and need to make some edits, Array.splice() comes in handy. The splice() method will let you INSERT or REMOVE elements from a javascript array.

Imagine we have the following array [0,1,2,3,5,6]. The following code example will insert the number 4 between the 3 and 5.

			
				var my_array = [0,1,2,3,5,6];
				my_array.splice(4, 0, 4);
				console.log('my_array');
				//[0,1,2,3,4,5,6];
PARAMETERS START, DELETE COUNT, INSERTION NOTE NOTHING GETS RETURNED, ACTS ON THE ORIGINAL ARRAY ['a','b','c','e','f'];
